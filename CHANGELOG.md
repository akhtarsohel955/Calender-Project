# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release of Wall Calendar application
- Interactive monthly calendar view with date selection
- Date range selection functionality
- Note-taking system with localStorage persistence
- Pin notes to specific dates or date ranges
- Smooth month navigation with flip animations
- Responsive design for mobile and desktop
- Paper-like aesthetic with spiral binding decoration
- Weekend highlighting
- Today indicator
- Note indicators on calendar dates

### Features
- Built with React 18 and TypeScript
- Tailwind CSS for styling
- Framer Motion for animations
- date-fns for date manipulation
- Radix UI components for accessibility

### Technical
- Vite build system
- ESLint configuration
- Vitest for unit testing
- Playwright for e2e testing
